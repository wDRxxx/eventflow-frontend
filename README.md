## How to run
1. Run EventFlow [backend](https://github.com/wDRxxx/eventflow-backend)
2. Configure .env file for your EventFlow [backend](https://github.com/wDRxxx/eventflow-backend)
3.
    ```shell
    npm run dev
    ```
